# Go BingAI Pass

## 环境变量

- `PORT`：API端口号，默认为 `7860`
- `PROXY_SERVER`: 代理服务器地址, 格式: `http://127.0.0.1:18080`
- `BROWSER_BINARY`: 浏览器二进制文件路径, 默认自动下载
- `PASS_TIMEOUT`: 通过人机验证等待时间
